<objective>
Let the user rename each exercise group on the /results dashboard with a persistent custom label.
Today every group shows the auto-generated "Exercício N". The user wants to click that label, type a
name (e.g. "C8-5_181 · C8 + C3 + C1"), press Enter, and have it stick across reloads.

Purpose: Make the research dashboard readable — exercise groups become meaningful, named records
instead of anonymous time windows.
Output: An `exercise_labels` table + model, a dedicated POST controller, a new route, and an inline
edit-in-place label in the exercise view of results.blade.php.
</objective>
